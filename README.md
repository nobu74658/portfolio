### 研究内容

物体検出を用いた研究をしています。
花の発生過程や進化を解明するために、花弁の発生をモデル化する研究が植物学の分野で行われています。
モデル化するためには、実在する花の花弁の並び方を分類する必要があり、その作業を自動化する研究をしています。
まず、花の画像から花弁同士の前後関係を検出して、花を花弁の並び方で分類します。（160 字）
